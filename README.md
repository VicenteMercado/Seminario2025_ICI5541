# Seminario2025_ICI5541
Proyecto de Seminario de Título: GENERACIÓN DE MAPAS A PARTIR DE NARRACIONES LITERARIAS

relaciones_espaciales.py se encarga de la extracción de las relaciones espaciales
solver_grafos.py se encarga de recibir estas relaciones extraidas y pasarlas por un solver
comparar_oficial_vs_solution.py se encarga de comparar el grafo generado por el modelo y pasado por el solver con el grafo oficial
official_nodes.csv Corresponde a las coordenadas puestas a mano de los lugares dentro de la ciudad de Luthadel.
